エアコン販買マスター Pro 正式版 2.0.1

【変更内容】
・実際のホーム画面を新デザインへ変更
・名前を大きく表示
・販売レベル、EXP、今日の進捗を表示
・販売モード／学習モードを大型カード化
・便利機能を8項目に整理
・iPhone／iPad対応
・既存の学習、接客、工事、図解機能は維持

【GitHubへの反映】
1. ZIPを展開
2. 中の全ファイルをGitHubへアップロード
3. Commit changes
4. 数分後に公開URLを再読み込み
5. 古い表示の場合はアプリを閉じ、Safariで再読み込み

index.htmlだけでなく、中の全ファイルをアップロードしてください。
