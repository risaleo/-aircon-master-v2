【更新手順】
1. GitHubで対象リポジトリを開く
2. Add file → Upload files
3. ZIPを展開した中身をすべて選択
4. Commit changes
5. Safariで公開URLを開き直す
6. 画面に「正式版 1.0.0」と表示されることを確認
